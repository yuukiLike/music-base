# 音程的分类 Interval Classification

## 按性质分类

音程可以按照其"听感"分为几大类：

### 完全协和音程 (Perfect Consonances)
- **纯一度** (Unison) — 0 个半音
- **纯八度** (Octave) — 12 个半音
- **纯五度** (Perfect 5th) — 7 个半音
- **纯四度** (Perfect 4th) — 5 个半音

这些音程听起来最"融合"，两个音几乎合为一体。

### 不完全协和音程 (Imperfect Consonances)
- **大三度** / **小三度** — 4 / 3 个半音
- **大六度** / **小六度** — 9 / 8 个半音

这些音程听起来悦耳，但两个音仍然各自可辨。和弦的"明亮"或"忧郁"感主要来自三度音程。

### 不协和音程 (Dissonances)
- **大二度** / **小二度** — 2 / 1 个半音
- **大七度** / **小七度** — 11 / 10 个半音
- **增四度/减五度** (Tritone) — 6 个半音

这些音程听起来有"紧张感"，需要"解决"到协和音程。

## 按度数分类

在传统乐理中，音程也用"度"来命名：

| 度数 | 包含的音名数 | 可能的变化形式 |
|------|-------------|---------------|
| 一度 | 1 | 纯一度、增一度 |
| 二度 | 2 | 小二度、大二度、增二度 |
| 三度 | 3 | 减三度、小三度、大三度、增三度 |
| 四度 | 4 | 减四度、纯四度、增四度 |
| 五度 | 5 | 减五度、纯五度、增五度 |
| 六度 | 6 | 小六度、大六度 |
| 七度 | 7 | 减七度、小七度、大七度 |
| 八度 | 8 | 减八度、纯八度、增八度 |
