# 音程应用 Interval Applications

## 音程与旋律

旋律是一系列连续的音程。不同风格的音乐倾向于使用不同的音程：

- **流行音乐**：大量使用二度和三度（步进为主），偶尔跳进
- **古典声乐**：跳进（四度、五度、六度）更常见
- **爵士**：七度和增四度创造特有的色彩

## 音程与和弦

和弦是多个音程的垂直叠加：

- **大三和弦** = 大三度 + 小三度
- **小三和弦** = 小三度 + 大三度
- **属七和弦** = 大三度 + 小三度 + 小三度

理解音程是理解和弦构造的基础。

## 音程与情感

| 音程 | 情感色彩 |
|------|----------|
| 大三度 | 明亮、快乐 |
| 小三度 | 忧郁、柔和 |
| 纯五度 | 空旷、开阔 |
| 小二度 | 紧张、悬疑 |
| 增四度 | 不安、神秘 |
| 纯八度 | 宏大、完整 |
