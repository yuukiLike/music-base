# 协和与不协和 Consonance & Dissonance

## 什么决定了协和与不协和？

音程的协和度与两个音的频率比有关。比值越简单，听起来越"融合"。

| 音程 | 频率比 | 协和度 |
|------|--------|--------|
| 纯一度 | 1:1 | 完全协和 |
| 纯八度 | 2:1 | 完全协和 |
| 纯五度 | 3:2 | 完全协和 |
| 纯四度 | 4:3 | 完全协和 |
| 大三度 | 5:4 | 不完全协和 |
| 小三度 | 6:5 | 不完全协和 |

频率比越复杂（如小二度 16:15），听起来就越"紧张"。

## 不协和不等于"难听"

不协和音程在音乐中不可或缺：

- **紧张感**创造了"解决"的期待 — 这是音乐前进的动力
- **小二度**在爵士和弦中增添色彩
- **增四度**（Tritone）被称为"魔鬼的音程"，但它是属七和弦的灵魂
- 现代音乐中，不协和音程被广泛使用来表达情感
