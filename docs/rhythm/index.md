# 强拍与弱拍

虽然 4/4 看起来只是 1 2 3 4 的重复，但听感上，它们通常不是完全平的。

## 4/4 的强弱规律

最粗略地说，在 4/4 里：

```
强  弱  次强  弱
 1   2    3    4
```

- 第 1 拍通常最稳、最强
- 第 3 拍也常常比较重要
- 第 2、4 拍通常相对弱一些

这会让同样的 4 个格子，开始出现起伏。

## 3/4 的强弱规律

```
强  弱  弱
 1   2   3
```

华尔兹的"蹦嚓嚓"感觉，就来自这个 `强-弱-弱` 的模式。

## 为什么这重要？

因为音乐不是只靠"时间切分"，还靠"重心分布"。

哪怕你只是把同样的音放在不同拍子上，感觉都会不同：

| 音落在 | 感觉 |
|--------|------|
| 第 1 拍 | 稳定、肯定 |
| 第 2 拍 | 经过、过渡 |
| 第 3 拍 | 次要重音 |
| 第 4 拍 | 向前推动，要"接下去" |

这就是"为什么固定骨架也有运动感"的开端。

## 切分音：打破预期

当重音故意不落在强拍上时，就产生了**切分音**（Syncopation）。

```
正常：  强 弱 次强 弱
切分：  弱 强 弱   强
```

切分音让音乐有"摇摆"和"不安分"的感觉。爵士、放克、拉丁音乐大量使用切分。
