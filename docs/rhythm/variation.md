# 节奏变化

时间骨架固定，不代表节奏只能机械重复。

## Rhythm / 节奏

节奏表示：**音符如何落在时间网格上**。

不是音有多高，而是：

- 什么时候响
- 响多久
- 中间是否停顿
- 是平均分布还是不平均分布

## 节拍和节奏不是一回事

这是高频混淆点。

| 概念 | 英文 | 本质 |
|------|------|------|
| 节拍 | Beat | 稳定脉冲，底层网格 |
| 节奏 | Rhythm | 事件如何落在网格上 |

你可以把它理解为：

- **beat** = 时间格子
- **rhythm** = 往格子里放事件的方法

## 一个最简单的对比

同样是 4/4，同样只有一个音 C：

```
方案 A：C  C  C  C     ← 每拍都响
方案 B：C  -  C  C     ← 第2拍空了
方案 C：C  C  -  C     ← 第3拍空了
方案 D：-  C  C  -     ← 两头空
```

拍号没变，速度没变，但节奏已经不同了。

**固定拍子之上，节奏仍然可以千变万化。**

## 常见节奏型

| 节奏型 | 模式 | 常见于 |
|--------|------|--------|
| 四拍均分 | `X X X X` | 行进、基础 |
| 后半拍强调 | `- X - X` | 雷鬼 |
| 附点节奏 | `X. x X. x` | 摇摆、复古 |
| 三连音 | `X x x X x x` | 布鲁斯、6/8 |

## 从节奏到旋律

到这里，你已经理解了音乐的时间维度：

1. ✅ 时间骨架（拍、速度、小节）
2. ✅ 重复性（骨架重复，内容不必重复）
3. ✅ 强弱拍（不是平的，有起伏）
4. ✅ 节奏变化（同样的格子，不同的放法）
