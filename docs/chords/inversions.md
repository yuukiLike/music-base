# 和弦转位 Chord Inversions

## 什么是转位？

和弦转位是指改变和弦中**最低音**的音。和弦的组成音不变，只是音的排列顺序不同。

以 C 大三和弦 (C-E-G) 为例：

| 转位 | 最低音 | 排列 | 记号 |
|------|--------|------|------|
| 原位 | C（根音）| C-E-G | C |
| 第一转位 | E（三音）| E-G-C | C/E |
| 第二转位 | G（五音）| G-C-E | C/G |

## 为什么要转位？

1. **声部进行更流畅** — 低音线条可以级进移动，而不是大幅跳跃
2. **改变和弦的"重量感"** — 不同转位有不同的稳定度
3. **创造低音旋律** — 经典的下行低音线：C → C/B → C/A → C/G

## 转位的稳定性

- **原位**：最稳定，适合乐句的开始和结束
- **第一转位**：较轻盈，适合经过性使用
- **第二转位**：最不稳定，在古典和声中有严格的使用规则
