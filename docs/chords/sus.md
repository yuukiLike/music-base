# 挂留和弦 Suspended Chords

挂留和弦用二度或四度音**替代**三度音，消除了大/小三和弦的"明亮"或"忧郁"感，创造一种**悬而未决**的效果。

## Sus2 挂二和弦

结构：根音 + **大二度** + 纯五度

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 |
| 二音 | D | 2 半音 |
| 五音 | G | 7 半音 |

听感：开放、空灵

## Sus4 挂四和弦

结构：根音 + **纯四度** + 纯五度

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 |
| 四音 | F | 5 半音 |
| 五音 | G | 7 半音 |

听感：有张力，强烈期待解决到大三或小三和弦

## 为什么叫"挂留"？

"Suspended"（悬挂）是因为四度音像是被"悬挂"在那里，等待"落下"（解决）到三度音。这种 sus4 → 大三 的解决是古典和声中最常见的装饰手法之一。
