## 使用方法

1. 选择一个**根音**（C, C#, D, ... B）
2. 选择**和弦类型**（大三、小三、七和弦等）
3. 点击 **▶ 播放和弦** 听效果
4. 点击钢琴键听单独的音
5. 尝试不同的组合，感受和弦色彩的变化

## 探索建议

- **对比大三和小三** — 只差一个半音，情感完全不同
- **试试属七和弦** — 感受它"需要解决"的张力
- **探索挂留和弦** — 没有三度音的"悬而未决"感
- **减七和弦** — 最紧张的和弦之一，戏剧化的效果
