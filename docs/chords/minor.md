# 小三和弦 Minor Triad

小三和弦听起来**忧郁、柔和、内省**，与大三和弦形成鲜明对比。

## 结构

小三和弦 = **根音 + 小三度 + 纯五度**

也可以理解为：根音 + 小三度 + 大三度

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 半音 |
| 三音 | D# | 3 半音（小三度）|
| 五音 | G | 7 半音（纯五度）|

## 大三 vs 小三：只差一个半音

大三和弦和小三和弦的唯一区别就是**三音**：
- 大三和弦：根音 + **4** 半音 + 7 半音
- 小三和弦：根音 + **3** 半音 + 7 半音

一个半音的差别，完全改变了和弦的情感色彩。
