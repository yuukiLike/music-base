# 七和弦 Seventh Chords

七和弦在三和弦的基础上加上一个**七度音**，从三个音变为四个音。这个额外的音赋予和弦更丰富的色彩。

## 常见七和弦类型

### 大七和弦 (Major 7th)
结构：大三 + 纯五 + **大七度** (11 半音)
- C 大七：C - E - G - B
- 听感：温暖、梦幻、爵士感
- 常见于：流行抒情、爵士、城市音乐

### 小七和弦 (Minor 7th)
结构：小三 + 纯五 + **小七度** (10 半音)
- C 小七：C - D# - G - A#
- 听感：柔和、忧郁但不沉重
- 常见于：爵士、R&B、Neo Soul

### 属七和弦 (Dominant 7th)
结构：大三 + 纯五 + **小七度** (10 半音)
- C 属七：C - E - G - A#
- 听感：有张力、需要解决
- 常见于：布鲁斯、摇滚、古典终止式

### 减七和弦 (Diminished 7th)
结构：小三 + 减五 + **减七度** (9 半音)
- C 减七：C - D# - F# - A
- 听感：极度紧张、戏剧化
- 特点：所有音等距（每 3 个半音），只有 3 种不同的减七和弦

### 半减七和弦 (Half-Diminished 7th)
结构：小三 + 减五 + **小七度** (10 半音)
- C 半减七：C - D# - F# - A#
- 听感：忧郁中带紧张
- 常见于：爵士 ii-V-I 进行
