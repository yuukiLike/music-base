# 大三和弦 Major Triad

大三和弦是最基本、最常见的和弦类型，听起来**明亮、稳定、快乐**。

## 结构

大三和弦 = **根音 + 大三度 + 纯五度**

也可以理解为：根音 + 大三度 + 小三度（两个三度叠加）

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 半音 |
| 三音 | E | 4 半音（大三度）|
| 五音 | G | 7 半音（纯五度）|

## 所有 12 个大三和弦

| 根音 | 和弦音 |
|------|--------|
| C | C - E - G |
| C# | C# - F - G# |
| D | D - F# - A |
| D# | D# - G - A# |
| E | E - G# - B |
| F | F - A - C |
| F# | F# - A# - C# |
| G | G - B - D |
| G# | G# - C - D# |
| A | A - C# - E |
| A# | A# - D - F |
| B | B - D# - F# |
