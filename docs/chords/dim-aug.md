# 减三和弦与增三和弦 Diminished & Augmented

## 减三和弦 (Diminished)

减三和弦 = **根音 + 小三度 + 减五度（6 个半音）**

听起来**紧张、不稳定**，强烈需要"解决"到其他和弦。

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 半音 |
| 三音 | D# | 3 半音（小三度）|
| 五音 | F# | 6 半音（减五度/增四度）|

减三和弦由两个**小三度**叠加而成。

## 增三和弦 (Augmented)

增三和弦 = **根音 + 大三度 + 增五度（8 个半音）**

听起来**梦幻、悬浮、不安定**。

| 从 C 构建 | 音符 | 音程 |
|-----------|------|------|
| 根音 | C | 0 半音 |
| 三音 | E | 4 半音（大三度）|
| 五音 | G# | 8 半音（增五度）|

增三和弦由两个**大三度**叠加而成。

## 四种三和弦对比

| 类型 | 三度 + 三度 | 五度 | 情感 |
|------|------------|------|------|
| 大三 | 大三 + 小三 | 纯五度 (7) | 明亮稳定 |
| 小三 | 小三 + 大三 | 纯五度 (7) | 忧郁柔和 |
| 减三 | 小三 + 小三 | 减五度 (6) | 紧张不稳 |
| 增三 | 大三 + 大三 | 增五度 (8) | 梦幻悬浮 |
