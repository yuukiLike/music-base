# 教会调式 Church Modes

教会调式是中世纪音乐的遗产，每种调式都有独特的情感色彩。它们都可以理解为**从大调音阶的不同音级开始**。

## 七种调式

以 C 大调的音符为基础（全部白键）：

| 调式 | 起始音 | 音符 | 情感色彩 |
|------|--------|------|----------|
| Ionian (伊奥尼亚) | C | C-D-E-F-G-A-B | = 大调，明亮稳定 |
| Dorian (多利亚) | D | D-E-F-G-A-B-C | 小调但更明亮，爵士 |
| Phrygian (弗里几亚) | E | E-F-G-A-B-C-D | 异域，西班牙/弗拉门戈 |
| Lydian (利底亚) | F | F-G-A-B-C-D-E | 梦幻，飘浮感 |
| Mixolydian (混合利底亚) | G | G-A-B-C-D-E-F | 摇滚，布鲁斯 |
| Aeolian (爱奥尼亚) | A | A-B-C-D-E-F-G | = 自然小调，忧郁 |
| Locrian (洛克里亚) | B | B-C-D-E-F-G-A | 极不稳定，几乎不用 |

## 如何理解调式？

最实用的方式是用**和大调/小调的对比**来记忆：

- **Dorian** = 自然小调 + 升高第 6 音 → "明亮的小调"
- **Phrygian** = 自然小调 + 降低第 2 音 → "异域的小调"
- **Lydian** = 大调 + 升高第 4 音 → "梦幻的大调"
- **Mixolydian** = 大调 + 降低第 7 音 → "摇滚的大调"

## 调式在现代音乐中的应用

- **Dorian**: Miles Davis "So What"、Daft Punk "Get Lucky"
- **Mixolydian**: Beatles "Norwegian Wood"、大量摇滚乐
- **Lydian**: 电影配乐中的"梦幻"场景
