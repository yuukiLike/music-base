# 五声音阶 Pentatonic Scales

五声音阶只有 **5 个音**，是全世界各文化中最广泛使用的音阶。中国传统音乐（宫商角徵羽）就是五声音阶。

## 大调五声音阶

去掉大调音阶中的第 4 和第 7 音（最"紧张"的两个音）。

音程模式：W-W-**W½**-W-**W½**

以 C 为例：C-D-E-G-A

### 与中国传统五声的对应

| 西方名称 | 中国名称 | C 大调五声 |
|----------|----------|-----------|
| 1 (主音) | 宫 | C |
| 2 | 商 | D |
| 3 | 角 | E |
| 5 | 徵 | G |
| 6 | 羽 | A |

## 小调五声音阶

大调五声音阶的关系小调版本。

以 A 为例：A-C-D-E-G

### 为什么五声音阶这么好用？

- **没有半音** — 任意两个音放在一起都不会"打架"
- **即兴神器** — 弹错的概率极低，非常适合初学即兴
- **跨文化** — 从中国民歌到布鲁斯到流行摇滚都在用
