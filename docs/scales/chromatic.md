# 半音阶 Chromatic Scale

半音阶包含一个八度内的**所有 12 个音**，每个音之间都是半音关系。

## 结构

H-H-H-H-H-H-H-H-H-H-H-H（12 个半音）

以 C 开始：C-C#-D-D#-E-F-F#-G-G#-A-A#-B-C

## 半音阶不是"音阶"

严格来说，半音阶更像是所有可用音的**完整清单**。其他所有音阶都是半音阶的**子集** — 从 12 个音中选取特定的几个。

| 音阶 | 从 12 个音中选 |
|------|---------------|
| 大调 | 7 个 |
| 五声音阶 | 5 个 |
| 布鲁斯音阶 | 6 个 |
| 全音阶 | 6 个 |
| 半音阶 | 全部 12 个 |

## 半音阶的用途

- **经过句** — 用半音连接两个音，创造流畅的旋律线
- **装饰音** — 在主要音符周围添加半音装饰
- **爵士即兴** — 在和弦音之间用半音"接近"（approach notes）
- **紧张感** — 快速的半音上行或下行创造戏剧效果
