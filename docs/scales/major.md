# 大调音阶 Major Scale

大调音阶是西方音乐最基础的音阶，你听到的大部分流行歌曲都基于大调。它听起来**明亮、积极、稳定**。

## 结构

大调音阶的音程模式：**全-全-半-全-全-全-半** (W-W-H-W-W-W-H)

以 C 大调为例：

| 音级 | 音名 | 到下一个音的距离 |
|------|------|----------------|
| 1 (主音) | C | 全音 |
| 2 | D | 全音 |
| 3 | E | **半音** |
| 4 | F | 全音 |
| 5 | G | 全音 |
| 6 | A | 全音 |
| 7 (导音) | B | **半音** |
| 8 (八度) | C | — |

## 为什么 C 大调特殊？

C 大调是唯一一个**全部由白键组成**的大调音阶。这使它成为学习的最佳起点 — 但所有大调的音程模式完全相同，只是起始音不同。
