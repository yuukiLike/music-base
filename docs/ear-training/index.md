# 听音训练

训练你的耳朵识别音程和和弦 — 这是音乐能力中最实用的部分。

## 训练建议

### 音程训练

- **简单模式**：先分清大三度和小三度（大三度明亮，小三度忧郁）
- **中等模式**：加入二度和六度，注意二度的"紧张感"
- **困难模式**：三全音和大七度是最容易混淆的

### 和弦训练

- **简单模式**：大三和弦 vs 小三和弦 — 一个阳光，一个阴郁
- **中等模式**：减三和弦有"不稳定"感，增三和弦有"悬浮"感
- **困难模式**：七和弦的差异更微妙，需要反复对比

### 通用技巧

1. **不要猜** — 听不出来就再播放一次
2. **错了就对比** — 点开正确答案听一遍，和你选的对比
3. **短时高频** — 每天 5 分钟比一周一次 1 小时有效
4. **先量后质** — 正确率 70% 以上再升难度
