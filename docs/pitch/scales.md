# 音阶与调

## Scale / 音阶

音阶表示：**按顺序排列的一组音**。

例如 C 大调音阶：

```
C  D  E  F  G  A  B  C
```

你现在可以把它看成：**一个常用可选音集合**。

不同的音阶 = 从 12 个半音中选不同的子集。

## 大调音阶：最基础的模式

大调的音程模式是：**全-全-半-全-全-全-半**

```
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
```

C 大调是唯一全部由白键组成的大调。但所有大调的音程模式完全相同，只是起始音不同。

## 小调音阶：忧郁的版本

自然小调的音程模式：**全-半-全-全-半-全-全**

```
A → B → C → D → E → F → G → A
  全  半  全  全  半  全  全
```

大调 → 明亮。小调 → 忧郁。区别只在音程模式不同。

## Key / 调

调比音阶更像"组织中心"。

- **音阶** = 哪些音常用
- **调** = 这些音围绕谁组织、谁最像"家"

"C 大调"意思是：用 C D E F G A B 这组音，C 是最稳定的"回家"音。

## 其他常见音阶

| 名称 | 音数 | 特点 |
|------|------|------|
| 大调 | 7 | 最基础，明亮 |
| 自然小调 | 7 | 忧郁版大调 |
| 五声音阶 | 5 | 中国传统（宫商角徵羽），即兴神器 |
| 布鲁斯 | 6 | 蓝调特有色彩 |
| 半音阶 | 12 | 所有音，完整清单 |
